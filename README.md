### 📥 데이터 다운로드 방법

1️⃣ **데이콘 대회 페이지에 접속하여 대회에 참여합니다.**  
2️⃣ **아래 데이터를 다운로드하세요.**
   - `train_input`: 흑백, 일부 손상된 학습용 PNG 이미지 (29,603장)
   - `train_gt`: 원본 학습용 PNG 이미지 (29,603장)
   - `test_input`: 흑백, 일부 손상된 평가용 PNG 이미지 (100장)
   - `train.csv`: 학습용 입력 이미지와 정답 이미지 경로 매핑
   - `test.csv`: 테스트용 입력 이미지 경로
